# Setting  up 


 - For linux users
 ```
    sudo apt-get install virtualenv

    virtualenv .venv

    source .venv/bin/activate

    touch  .env 

    echo "KEY=YOUR_KEY" >> .env
    
    echo "LOCATION=YOUR_LOCATION" >> .env

    echo "ENDPOINT=YOUR_ENDPOINT" >> .env

```
